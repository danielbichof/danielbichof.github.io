# danielbichof.github.io.hugo

Hugo migration of `danielbichof.github.io.old` using `geekifan/hugo-theme-chirpy`.

## Run (Docker)

### Development

```bash
docker compose up dev
```

Open: <http://localhost:1313>

### Production Preview

```bash
docker compose up --build web
```

Open: <http://localhost:8080>
